### 目前研究思路
  针对身体姿态估计问题，需要网络处理时序信息，因此传统的卷积网络无法实现，需要一些能处理时序信息的网络；
  
  目前较为前沿的能够处理时序信息和视频数据的网络有3D-CNN、MViT、ConvLSTM、YOLO based等；
  
  上面几种主流的网络模型都是复杂的高效模型，因此可以选择某一模型进行调整，然后作为教师模型进行知识蒸馏；
  
  选择MobileNet作为学生模型，以达到轻量化目的

  目前已找到一个交警交通手势的视频数据集吗，共有8种动作，每个视频时长平均8min
